## Install neovim

```shell
brew install neovim
```

## Activate Clipboard

```shell
sudo apt-get install wl-clipboard
```

```
vim.opt.clipboard = "unnamedplus"
```

## Treesitter Requirement

```shell
brew install tree-sitter-cli
```

## Install acpi battery provider

```
sudo apt install acpi
```

## Install rg

```shell
brew install rg
```

## Install LazyGit

```shell
brew install lazygit
```
